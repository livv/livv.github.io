vJSON

Developer Tools

A Safari extension for macOS that turns raw JSON pages into a foldable, syntax-highlighted tree—with toolbar controls, nested JSON expansion, and quick copy.

Key Features

Syntax & structure

VS Code–style theme, indentation guides, and key/value counts so large payloads stay readable.

Fold & expand

Expand or collapse all, fold to a chosen depth, and toggle word wrap from the toolbar.

Nested JSON strings

Detect embedded JSON in string values and expand them inline without leaving the page.

URLs & copy

Clickable links for detected URLs and one-click copy for the full JSON body.
